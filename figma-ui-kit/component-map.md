# Component Map

## Boards in `figma-ui-kit.svg`

- `TOKENS`: color swatches, spacing scale, radius scale
- `TYPOGRAPHY`: display, heading, body, mono samples
- `COMPONENTS`: buttons, badges, inputs, cards, top bar
- `PATTERNS`: table block, status panel, error panel

## Variant properties

- `Size=Sm|Md|Lg`
- `State=Default|Hover|Active|Disabled`
- `Theme=Dark`

## Constraints and auto layout

- Horizontal containers: `Space between`, center align
- Vertical cards/panels: `Packed`, 8/12/16 spacing
- Table row: fixed height + fill container width
